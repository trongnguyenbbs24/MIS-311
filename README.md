# MIS-311
Data Analysis and Insight Project for MIS 311

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on an Electric Vehicle dataset.

## Objectives
- Clean and preprocess data
- Perform descriptive statistics
- Identify trends and insights
- Create visualizations

## Tools Used
- Microsoft Excel
- GitHub

## Key Insights
1. Tesla dominates the dataset with the highest number of electric vehicles.
2. Electric vehicle adoption increased significantly in recent model years.
